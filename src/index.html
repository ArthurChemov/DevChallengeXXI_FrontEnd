Data Visualization Tool Documentation

Main Features

  • Header: Displays the tool name and a theme switch button.
  • File Upload: Supports drag & drop, file selection, and manual data entry (CSV, Excel, JSON).
  • Preview Data: After uploading data the modal window appears with the data in a table and a close button
  • Chart Type: Choose from line, bar, or pie charts.
  • Generate Chart: Button to create the chart.
  • Reset: Button to clear inputs and reload.

Visualization Area

  • Chart Display: Main area for viewing charts.
  • Customization: Adjust titles, axis labels, colors, and styles.

Export Options

  • Formats: Export as PNG, PDF, or SVG.
  • Print: Option to print the chart directly or via Ctrl + P.
